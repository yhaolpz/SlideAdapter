# SlidingMenu
动手打造史上最简单的 Recycleview 侧滑菜单

![效果图](https://raw.githubusercontent.com/yhaolpz/SlidingMenu/master/img-folder/ezgif-4-4f5325558d.gif)

第三方库的局限性：

 1.菜单文字一旦确定将无法修改
 2.侧滑时整个 item 都会滑动
 3.无法自定义菜单样式
 
为了弥补这些缺陷
 
  http://blog.csdn.net/yhaolpz/article/details/77366154

提供一种 Recycleview 侧滑菜单实现方案

优点： 小巧、灵活、简单以及简单。

  2.解决多指触摸将同时打开多个item
